How it works

A simple three-step process that keeps teams in control.

1 — Define schema

Model entities, pages, permissions and workflows as JSON files stored in Git.

2 — Platform engine

A runtime interpreter auto-generates the UI, logic and API interactions from schemas.

3 — Instant SPA

The SPA updates immediately — no code, no rebuild, no downtime.

Example schema (excerpt)

{
  "collection": "clients",
  "fields": [
    {"name":"firstName","type":"string"},
    {"name":"lastName","type":"string"},
    {"name":"email","type":"string","validators":["email"]}
  ],
  "permissions": {
    "list": "role:admin or role:manager",
    "edit": "role:admin"
  }
}
